El propósito de este juego es sobrevivir a la horda de bugs que se aproximarán para destruir el mundo del software. Para lograr sobrevivir sigue las siguientes instrucciones y seguramente lograrás la victoria!

Utiliza la tecla 'UP ARROW' para mover a Jack hacia arriba.

Utiliza la tecla 'LEFT ARROW' para mover a Jack hacia la izquierda.

Utiliza la tecla 'RIGHT ARROW' para mover a Jack hacia la derecha.

Utiliza la tecla 'DOWN ARROW' para mover a Jack hacia abajo.

Utiliza la tecla 'UP' para mover a Jack hacia arriba.
